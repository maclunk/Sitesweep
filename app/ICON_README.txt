Favicon-Placeholder

Für Next.js App Router werden folgende Dateien benötigt:

1. app/icon.png (512x512px empfohlen) - wird automatisch von Next.js erkannt
2. public/favicon.ico (32x32px) - als Fallback

Diese Dateien können mit Online-Tools erstellt werden:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

Bitte erstellen Sie diese Dateien und ersetzen Sie diese Platzhalter-Datei.
