# Vorher-Nachher Screenshots

Dieser Ordner enthält die Screenshots für den Vorher-Nachher-Slider auf der Landingpage.

## Benötigte Dateien

Bitte legen Sie hier zwei Screenshot-Dateien ab:

- `website-before.png` oder `website-before.jpg` - Screenshot der Website vor der Optimierung
- `website-after.png` oder `website-after.jpg` - Screenshot der Website nach der Optimierung

**WICHTIG:** Falls Sie `.jpg` oder `.jpeg` Dateien verwenden, müssen Sie die Pfade in `app/page.tsx` entsprechend anpassen:
- Von: `/images/before-after/website-before.png`
- Zu: `/images/before-after/website-before.jpg` (oder `.jpeg`)

## Anforderungen

- **Format**: PNG (empfohlen) oder JPG/JPEG
- **Größe**: Empfohlene Breite: 1200-1920px
- **Höhe**: Sollte für beide Bilder identisch sein
- **Qualität**: Hochwertige Screenshots für beste Darstellung

## Verwendung

Die Screenshots werden automatisch im `BeforeAfterSlider`-Component verwendet, das auf der Landingpage (`app/page.tsx`) integriert ist.
